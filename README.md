# TwinCAT 3 Traffic Light Intersection

A professional PLC portfolio project implementing a simulated traffic light intersection using **TwinCAT 3**, **Structured Text**, **object-oriented programming**, **test-driven development**, and a simple **HMI/GUI**.

The goal of this project is not only to control traffic lights, but to demonstrate clean PLC software architecture, modular design, safety thinking, simulation, testing, and documentation.
